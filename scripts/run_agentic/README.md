# Agentic Facebook CLI (`scripts/run_agentic/`)

Entrypoint scripts for the opt-in agentic flow (`ENABLE_AGENTIC_FACEBOOK_SYNC`). Run from the **repository root** so `.env` and imports resolve:

```bash
.venv/bin/python scripts/run_agentic/run_agentic_facebook_once.py --help
```

Documentation: [`agentic_facebook/README.md`](../../agentic_facebook/README.md) and [`agentic_facebook/docs/`](../../agentic_facebook/docs/).
